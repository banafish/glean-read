# 规范：AI 大纲合成与知识树 (ai-synthesis-tree)

## 概述
将“点”连成“线与面”。借助 LLM 总结标签分类下未组织的碎片，合成系统化知识脉络大纲，并挂载到常青知识树。

## 功能要求
1. **圈定合成集**: 在大屏幕前对某一特定标签内圈选的一组碎片，执行 `AI 大纲提炼` 动作。
2. **Prompt与合并**: 后端拼接碎片数组及用户批想请求 LLM。大纲输出需具备嵌套层级，且在每条提炼点标注引申自哪几个原始片段。
3. **结构化双向关联**: 生成的系统化提纲成为树状结构节点（Node），原始知识碎片作为叶属性挂载。
4. **树形数据与UI**: 前端提供管理、重命名、拖拽整棵知识大树的能力平台。

## API 与交互协定
- `POST /api/v1/knowledge/synthesis`: 发送圈选项进行大纲合成提取业务，可能为异步响应以防止 LLM 超时。
- **实体设计**: 定义树状关联模型，如引入 `parent_id` 构建层级的专题树体系。
